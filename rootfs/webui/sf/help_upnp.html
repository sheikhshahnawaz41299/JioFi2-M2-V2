• Setting the UPnP

The Universal Plug and Play (UPnP) service realizes intelligent connection between two UPnP devices using port forwarding. UPnP devices can obtain IP addresses automatically and access the Internet dynamically.


Procedure

  1. 1. Choose Setting > Gaming & Apps> UPnP.
  2. 2. Select:
    • Enable: to enable the UPnP service.
    • Disable: to disable the UPnP service.
  3. Click APPLY button.

Post requisites

A computer that runs the Windows XP operating system is used as an example to describe how to enable the UPnP service on a client.

  1. 1. Choose Start > Control Panel > Add or Remove Programs > Add/Remove Windows Components.
  2. 2. Select Networking Services and click Details.
  3. 3. Select UPnP User Interface to enable the UPnP service.
  4. 4. Click OK.

  • Port Forwarding

It allows remote computers (for example, computers on theÂ Internet) to connect to a specific computer or service within a private local-area network.

  1. 1. Choose Settings > Gaming & Apps > Port Forward.
  2. 2. Select the check box on the left of Enable Port Forward.
  3. 3. Select the user info from user list, that Host name and MAC address will add to port forwarding list, but you still have to input Config Name, Local and Wan Port by manually. Or you can input all of info by manually.
    Note:

    Local Port can be repeated, WAN port cannot be repeated.

  4. 4. Click APPLY button.
  5. 5. Start the server on the device that added the port forwarding list.
    Note:

    The server include Web server, ftp server, Web camera server etc.

  6. 6. Access the device started server with the other terminal connected Internet through WAN IP address with WAN port to the device started server. Such as: http://183.42.114.82:8080
  7. 7. The server can be accessed successfully.

  • How to add a port forwarding rule?

Procedure:

  • Record wan IP of your M2 from Web GUI
  • Choose IP address you want the M2 to forward to, you can get the IP info either from CMD terminal
    or just simply choose from User list on 'Gaming & Apps Management' page.
  • Go to Setting -> Gaming & Apps Management page, Enable Port Forward function
  • Fill in all fields for your port forwarding rule
    1. Click the 'ip' which you want to forward to, it will fill the Mac filed automatically.
    2. Fill Config name (rule name) and Local port/WAN port
  • Click "Apply" button to apply
  • Now the UDP traffic reach M2 WAN IP(ex: 10.72.125.90)/port 3560 will be redirect to IP of your PC(ex: 192.168.1.101/port 3500).
    (In this example, 3500 is 'Local port' and 3560 is 'Wan port')
