    PIN

    PIN Settings provides PIN security settings to prevent unauthorized use of the USIM card.

    Turning On or Off PIN Verification

    If PIN verification is turned on, you need to enter the PIN only when the M2 is turned on. If PIN verification is turned off, no PIN is required.

    Choose Settings > LTE > PIN , you can enable or disable the PIN verification.

    Context

    • If a PIN is required, enter the correct PIN.
    • If you enter your PIN incorrectly three times consecutively, the USIM card is locked and you must enter the PIN unlock key (PUK).
    • If you enter the PUK incorrectly ten times consecutively, the USIM card is locked permanently. If you cannot enter the correct PIN or PUK, network-associated functions are unavailable.
    • The PIN and the PUK are provided with the USIM card. If you did not receive them or forgot them, contact your service provider.

    Procedure

    1. Click Setting > LTE > PIN Setting.
    2. From PIN operation, select:
      • Enable: to turn on PIN verification.
      • Disable: to turn off PIN verification.
    3. Enter the correct PIN.
    4. Click APPLY button.
    Changing the PIN

    If PIN verification is turned on, you can change the PIN.

    Procedure

    1. Click Setting > LTE > PIN Setting.
    2. From PIN operation, select Change.
    3. Enter the current PIN.
    4. Enter a new PIN.
    5. Click APPLY button.

    Network

    MAC addresses filter

    You can control and manage the clients that access the wireless local area network (WLAN) to improve the security performance of the WLAN.

    1. Choose Settings > Network > MAC addresses filter
    2. From MAC Filter Mode to select a Media Access Control (MAC) filtering mode.
      • Disable: To disable the function of filtering MAC addresses.
      • Enable > Allow : If the MAC address of a client is listed in MAC address, the client is allowed to connect to the M2 over the WLAN.
      • Enable > Deny : If the MAC address of a client is listed in MAC Address, the client is prohibited from connection to the M2 over the WLAN.
    3. Fill in the MAC addresses of the clients from user list or manually.
    4. Click APPLY button.

    Note: any filtering mode to be selected, please make sure yourself can through the filter.

    SSID Broadcast

    If the service set identifier (SSID) broadcast function is enabled, the M2 broadcasts the SSID of the wireless local area network (WLAN) facilitating clients' access to the WLAN. The disadvantage is that unauthorized clients can also access the WLAN. If the SSID broadcast is disabled, the M2 does not broadcast the SSID of the WLAN. So the SSID must be entered manually and correctly from a client before the client accesses the WLAN. This improves the security of the WLAN.

    Procedure

    1. Click Setting > Wi-Fi.
    2. Set SSID Broadcast.
      • Enable: to enable the SSID broadcast.
      • Disable: to disable the SSID broadcast.
    3. Click APPLY button.

    Note: During the process of setting up the WLAN,you can set the SSID broadcast to Enable to facilitate the access from clients.After the WLAN is set up, set the SSID broadcast to Disable to improve the security of the WLAN.

    UPnP

    The Universal Plug and Play (UPnP) service realizes intelligent connection between two UPnP devices using port forwarding. UPnP devices can obtain IP addresses automatically and access the Internet dynamically.

    Procedure

    1. Choose Setting > Gaming & Apps Management > UPnP.
    2. Select:
      • Enable: to enable the UPnP service.
      • Disable: to disable the UPnP service.
    3. Click APPLY button.

      4. Post requisites

      A computer that runs the Windows XP operating system is used as an example to describe how to enable the UPnP service on a client.

      1. Choose Start > Control Panel > Add or Remove Programs > Add/Remove Windows Components.
      2. Select Networking Services and click Details.
      3. Select UPnP User Interface to enable the UPnP service.
      4. Click OK.
      Port Forwarding

      It allows remote computers (for example, computers on the Internet) to connect to a specific computer or service within a private local-area network.

      1. Choose Setting > Gaming & Apps Management > Port Forward.
      2. Select the check box on the left of Enable Port Forward.
      3. Select the user info from user list, that Host name and MAC address will add to port forwarding list, but you still have to input Config Name, Local and Wan Port by manually. Or you can input all of info by manually.

        Note: Local Port can be repeated, WAN port cannot be repeated.

      4. Click APPLY button.
      5. Start the server on the device that added the port forwarding list.

        Note: Local Port can be repeated, WAN port cannot be repeated.

      6. Access the device started server with the other terminal connected Internet through WAN IP address with WAN port to the device started server. Such as: http://183.42.114.82:8080
      7. The server can be accessed successfully.

      FAQs

      Clients Cannot Access the Internet Properly

      Procedure:

      1. Check that clients are connected to the M2 properly.
      2. Check that the M2 is powered on properly.
      3. Check that the M2 is in a location covered by communication networks and that the signal strength is strong.
      4. When the M2 is used for Internet access, the Point-to-Point Protocol (PPP) user name and password must be set correctly. Check that the PPP user name and password are correct. For details, see Setting > LTE > Default PDN Connection.
      5. Check that the network adapters of the clients are working properly.
      6. Contact your service provider if the problem persists.

      Clients Cannot Access the WLAN Properly

      Procedure:

      1. Check whether there are potential interference sources or shielding objects near the M2. If any, adjust the placement of the M2
      2. Check and record the settings of the following parameters on the clients and the M2: SSID, Encryption mode, and security key. The encryption mode and the security key on the clients must match those set on the M2. Otherwise, change the setting on the clients
      3. If the Media Access Control (MAC) address filtering in the wireless local area network (WLAN) is enabled on M2, check that the MAC addresses of the clients are not in the Black list. For details, see Settings > Network > MAC addresses filter .

      Login username and password is forgotten

      1. Long press Reset button for 3 seconds to restore the default setting
      2. Enter the default username and password both are admin to login web management.

      Wi-Fi Key Is Forgotten

      • Log in to the management page and obtain the current Wi-Fi key. For details, see Setting the WLAN Key.
      • Long press Reset button for 1 seconds to restore the default settings. Check the relevant label on the M2 to obtain the default Wi-Fi key.

      Multi-Network IP Address Conflict Occurs

      When multiple types of Internet connections are used simultaneously on the client, such as physical network adapter and Wi-Fi, inserting the M2 into the client may result in IP address conflict across multiple networks.

      Solution: Disable other network devices or disconnect other Internet connections from the client, and then restart M2.

      M2's Data Services Are Blocked by Some Antivirus Software

      Some antivirus software (for example, Bitdefender 2010) will block the M2's RNDIS port.

      Solution: Upgrade the relevant antivirus software to its latest version. If the problem persists, disable the software's firewall.

      What Can I Do If the Network Connection Is Frequently Interrupted?

      Procedure:

      1. Check the network signal strength. If the signal is weak, move the M2 somewhere with better reception.
      2. Visit other websites to check whether the issue lies within a particular site.
      3. Close all other applications that require network connections.
      4. Contact your Internet service provider if the problem persists.

      What Can I Do If I Cannot Open Web Pages After a Network Connection Is Set Up?

      Procedure:

      1. Re-set up a dial-up Internet connection.
      2. Check that the computer is not infected with viruses.
      3. Check that the APN user name and password in the profile management settings are correct. For details, contact your Internet service provider.
      4. Check that network connection has not been blocked by a firewall.
      5. Restart your computer, and re-set up a dial-up Internet connection.

      What Can I Do If the Icons That Indicate No Signals and No Service Are Displayed?

      Procedure:

      1. Check that the M2 is in an area with good reception. For details, contact your Internet service provider.
      2. Check that you have subscribed to network services for your USIM card, and that there is no outstanding balance. For details, contact your Internet service provider.

      What Can I Do If the Network Access Rate Is Low?

      Procedure:

      1. Check that the M2 is in an area with good reception. For details, contact your Internet service provider.
      2. Check the network standard used by the Mobile Wi-Fi. The available network access rate varies by the network standard. For details, contact your Internet service provider.

      Cannot login Web management Page

      • Disconnect other network connections, such as an Ethernet or Wireless Fidelity (Wi-Fi) connection.
      • Clear the cookies and caches of IE browsers.
      • Close all IE browsers and Open again.
      • Check the name and password of administrator is correct.
      • Remove and then insert the M2, or restart the computer.

      How to add a port forwarding rule?

      Procedure:

      • Record wan IP of your M2 from Web GUI
      • Choose IP address you want the M2 to forward to, you can get the IP info either from CMD terminal
        or just simply choose from User list on 'Gaming & Apps Management' page.
      • Go to Setting -> Gaming & Apps Management page, Enable Port Forward function
      • Fill in all fields for your port forwarding rule
        1. Click the 'ip' which you want to forward to, it will fill the Mac filed automatically.
        2. Fill Config name (rule name) and Local port/WAN port
      • Click "Apply" button to apply
      • Now the UDP traffic reach M2 WAN IP(ex: 10.72.125.90)/port 3560 will be redirect to IP of your PC(ex: 192.168.1.101/port 3500).
        (In this example, 3500 is 'Local port' and 3560 is 'Wan port')
